# Originals / 原文库

This directory preserves public-safe original source material for *哲学马拉松 / Philosophy Marathon*.

The repository should not become a raw chat dump, but distilled philosophical chapters and Agent-readable files need source traceability. Save safe source material here first, then link distilled outputs back to it.

## What Belongs Here

- Public-safe dialogue excerpts selected for book development
- User-provided notes and source fragments related to self, consciousness, time, memory, AI, systems, and human-AI co-thinking
- Draft fragments before they are rewritten into `book/` or `human/`
- Source notes that explain where a claim, concept, or question came from

## What Does Not Belong Here

- Raw private conversations
- Passwords, tokens, API keys, private keys, account credentials
- Precise addresses, IDs, bank information, or other sensitive personal details
- Raw legal, health, family, or private AI memory content
- Unreviewed material that is not safe for a public repository

# Title

## Original Text / 原文

Preserve the public-safe original wording here.

## Source Notes / 来源说明

Record where this came from and what was removed or sanitized.

## Links to Distilled Outputs / 提炼后去向

List related `book/`, `human/`, `agent/`, or `data/` files.

## Rule

Original source preservation comes before distillation. If a future Agent creates a chapter idea, concept, claim, question, or dialogue index entry from source material, it should preserve or cite the source here first, unless privacy rules forbid raw storage.
